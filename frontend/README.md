# ShareBite: Frontend

The React application for the ShareBite food donation platform.

## Features
- **Donor Dashboard**: Create donations and view their status.
- **NGO Dashboard**: Match with nearby donations and track incoming deliveries.
- **Volunteer Dashboard**: Discover nearby pickup tasks and perform deliveries.
- **Admin Dashboard**: Oversee system activities and manage users.
- **Maps Integration**: Location-aware interactions using Leaflet.js.
- **Live Location Tracking**: Real-time map updates via Socket.io.

## Tech Stack
- React
- Vite
- Tailwind CSS (Layout & Styling)
- React Router (Routing)
- Axios (API requests)
- Socket.io-client (Real-time updates)

## Installation & Running
1. `npm install`
2. `npm run dev`
